PrepRadii App

PrepRadii is a peer-learning app that connects students by location and exam, enabling easy and instant collaboration.

1. Connect & Learn – Find nearby learners
2. Chat Instantly – Discuss, share ideas, and clear doubts
3. Collaborate & Grow – Form study groups

Tools & Technologies

Built using:

  • React Native
  • Expo
  • JavaScript
  • Firebase
  • BaaS
  • VS Code
